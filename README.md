### 🧩 Stack

- **Frontend** → React  
- **Backend** → Node.js (Express)  
- **Database** → MongoDB  
- **Reverse Proxy** → Nginx  
- **Containers** → Docker / Docker Compose  
- **Cloud** → AWS  
- **IaC** → Terraform  
- **CI/CD** → Build → Push → Deploy
